# Assignment 1 – Personal Portfolio Website

# Description
A fully responsive personal portfolio website built with pure HTML5 and CSS3 (no frameworks).

# Features
- Sticky navbar with smooth scroll & active link highlighting.
- Hero section with animated avatar ring and floating skill badges.
- About section with skill tags.
- Projects grid (featured card + regular cards).
- Contact section with info and functional form.
- Fully responsive.

# Structure
assignment1_portfolio/
    ─ index.html      ← Full HTML with semantic structure
    ─ style.css       ← All styles (CSS variables, responsive, animations)
